# Project Assignment 1

A React-based web application for project and task management with user authentication.

## Features

- User registration and login
- Dashboard overview
- Project management
- Task tracking
- Responsive UI

## Installation

1. Clone the repository:
   ```bash
   git clone <your-repo-url>
   cd projectAssing-1
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm start
   ```

The app will run on `http://localhost:3000`.

## Deployment

This project is configured for deployment on Vercel. Push your code to GitHub and connect the repository to Vercel for automatic deployment.

## Technologies Used

- React
- JavaScript
- CSS
- Context API for state management
